# SBI Customer Transaction Analysis System

## Overview
This project analyzes SBI customer transaction data and creates visualizations and dashboards.

## Features
- Data Cleaning
- Data Analysis
- Visualization
- Interactive Dashboard

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Streamlit

## Run Commands

```bash
python src/data_cleaning.py
python src/analysis.py
python src/visualization.py
streamlit run dashboard/app.py
